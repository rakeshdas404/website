Movie Rating Prediction

A Robust Pipeline for Personalized Recommendations with Random Forest Regressor.

VIEW REPORT

Project Overview

This project focuses on building a machine learning-based recommendation system to predict movie ratings for user-movie pairs, enabling personalized recommendations. The primary objective was to create a pipeline that generalizes well to unseen data, addressing cold-start problems (users or movies not present in training data) while optimizing model performance through feature engineering and hyperparameter tuning. The dataset was enriched with user-level and movie-level features, including genre-match scores, and categorical attributes were transformed using one-hot encoding. Missing values were carefully imputed, and outliers were retained to preserve data variability and user preferences.

The Random Forest Regressor was selected as the core model due to its ability to handle large datasets and non-linear relationships. The model was fine-tuned using GridSearchCV, exploring 162 hyperparameter combinations to maximize accuracy. Performance evaluation was conducted with metrics such as Mean Squared Error (MSE), Mean Absolute Error (MAE), and R² scores. Residual analysis confirmed unbiased predictions with a symmetric error distribution, ensuring no signs of overfitting or underfitting. The project successfully achieved a Test Set RMSE score of 89 (measured in hundredths of a star) on the leaderboard, demonstrating its reliability for practical recommendation tasks.
