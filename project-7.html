AI-Powered Smart Blind Stick

Combining Obstacle Detection, Emotion Recognition, and Adaptive Feedback for Visually Impaired Users

VIEW REPORT

Project Overview

This project proposes an AI-powered smart blind stick designed to improve mobility, safety, and independence for visually impaired individuals. It integrates advanced sensing technologies and AI algorithms to address the limitations of traditional assistive devices. The system employs real-time object detection using YOLOv8 models and Support Vector Machines (SVMs) for accurate obstacle recognition. Additionally, a facial emotion recognition module enhances user interaction by analyzing emotional states and providing personalized feedback via a Text-to-Speech (TTS) system. This combination of features enables the device to deliver context-aware responses that not only guide users through physical obstacles but also support their emotional well-being in challenging environments.

The solution incorporates dynamic adaptability for different scenarios, such as crowded streets or indoor spaces, using environmental sensors and gesture detection for non-verbal commands. By leveraging machine learning techniques like Histogram of Oriented Gradients (HOG) and deep neural networks, the system achieves high accuracy in detecting obstacles and classifying emotions. Evaluation results demonstrate 97.22% accuracy for object detection and 71.88% training accuracy for facial emotion recognition, proving its feasibility as a reliable assistive tool. The project highlights a scalable approach to assistive technology, making it accessible for resource-constrained settings and paving the way for future enhancements like GPS integration and voice command support.
