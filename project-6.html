Nonogram Puzzle Solver and Generator

Efficient Constraint-Based Solvers and Image-to-Nonogram Conversion

VIEW REPORT

Project Overview

This project tackles the problem of solving and generating Nonogram puzzles, a type of logic-based grid puzzle, using Python. The first part focuses on building a Nonogram Solver capable of parsing input files containing row and column clues, setting up a grid, and applying dynamic programming techniques to find valid solutions. The solver incorporates constraint propagation to refine possibilities, ensuring optimal solutions even for larger grids. The algorithm is optimized with timeout handling and file parsing enhancements, significantly improving scalability. While the solver successfully handles medium-sized puzzles, larger puzzles posed challenges, where partial solutions were generated as intermediate outputs, demonstrating the algorithm’s effectiveness and scalability.

The second part focuses on a Nonogram Generator, which converts images into solvable Nonogram puzzles. A threshold-based approach is used to preprocess input images, converting them to binary grids based on pixel intensity. Preprocessing steps include grayscale conversion, histogram equalization, and image resizing to ensure clarity and usability. The generator outputs both row and column clues and a visual representation of the grid, enabling users to recreate the puzzle manually. This implementation highlights the balance between image simplification and puzzle complexity, ensuring the generated puzzles are both solvable and visually meaningful.
